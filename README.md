# Aplikasi Stok Gudang Elektronik

[](https://tailwindcss.com)

[cite_start]Aplikasi Stok Gudang Elektronik adalah sistem manajemen inventaris berbasis web yang dirancang untuk mengoptimalkan pengaturan dan pengawasan persediaan barang di gudang[cite: 169]. [cite_start]Sistem ini memungkinkan pencatatan barang masuk dan keluar secara akurat serta menghasilkan laporan stok otomatis[cite: 170, 171].

## 📋 Fitur Utama

### Admin Gudang

  * [cite_start]**Manajemen Data**: Mengelola kategori barang, data barang, dan data supplier[cite: 162, 173].
  * [cite_start]**Transaksi**: Mencatat dan memanipulasi data barang masuk serta barang keluar[cite: 162].
  * [cite_start]**Pelaporan**: Mengunduh laporan stok dalam format Excel dan melihat riwayat transaksi[cite: 162, 174].

### Manajer

  * [cite_start]**Monitoring**: Mengakses dashboard untuk memantau pergerakan stok melalui grafik[cite: 176].
  * [cite_start]**Laporan**: Melihat dan mengunduh laporan stok tanpa hak akses untuk mengubah data[cite: 162, 176].

## 💻 Instalasi Lokal

Pastikan Anda sudah menginstal PHP (\>= 8.x), Composer, dan Node.js di perangkat Anda.

1.  **Clone repositori:**

    ```bash
    git clone https://github.com/username/stok-gudang-elektronik.git
    cd stok-gudang-elektronik
    ```

2.  **Instal dependensi:**

    ```bash
    composer install
    npm install && npm run dev
    ```

3.  **Konfigurasi Environment:**
    Salin file `.env.example` menjadi `.env` dan sesuaikan pengaturan database Anda.

    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4.  **Migrasi Database:**

    ```bash
    php artisan migrate --seed
    ```

5.  **Jalankan Server:**

    ```bash
    php artisan serve
    ```
